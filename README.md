# PD-Driver
<img width="385" height="382" alt="image" src="https://github.com/user-attachments/assets/a41bf2fa-18c3-42cb-8154-01e7db19a01b" />






A portable, efficient power delivery (PD) driver system designed to control a NEMA 17 stepper motor with precision and reliability. This project focuses on integrating a compact power supply with a high-performance stepper driver to enable smooth motion control for robotics, CNC machines, and automation applications.
